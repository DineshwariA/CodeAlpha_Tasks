# CodeAlpha Python Programming Internship

This repository contains Python programs completed as part of the
**CodeAlpha Python Programming Internship**. The tasks focus on building basic Python programming skills through beginner-friendly projects.

---

## Tasks Completed

### 1. Basic Chatbot
A simple rule-based chatbot that responds to predefined user inputs.

**Concepts Used:**
- Input / Output
- if-elif-else conditions
- while loop
- String handling

**File name:** CodeAlpha_BasicChatbot.py

---

### 2. Hangman Game
A text-based Hangman game where the user guesses a randomly selected word
one letter at a time with limited attempts.

**Concepts Used:**
- Lists
- Strings
- Random module
- Loops
- Conditional statements

**File name:** CodeAlpha_HangmanGame.py

---

### 3. Stock Portfolio Tracker
A program that calculates the total investment value using predefined
stock prices and user input.

**Concepts Used:**
- Dictionary
- User input
- Arithmetic operations
- Conditional statements

**File name:** CodeAlpha_StockPortfolioTracker.py
